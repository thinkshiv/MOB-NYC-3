# Lesson 04 - Intro to Programs and Functions

## Lesson Objectives

* Review variables, values, and types
* Utilize control flow to create a simple program flow in playgrounds.
* Apply Optionals and explain when and how to use them.


## Lesson Topics

* Review variables
* Control Flow
    * Review of Bool
    * Bools + Conditionals
        * if/else statements
    * Conditionals + Loops
        * while loops
        * for loops
* Optionals + nil
    * nil
    * using Optionals
